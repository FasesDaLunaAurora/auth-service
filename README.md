# Auth Service

Microserviço responsável por autenticação e autorização.

## Objetivo

Centralizar autenticação e autorização em aplicações modernas.

## Tecnologias

- FastAPI
- PostgreSQL
- JWT
- SQLAlchemy
- Alembic

## Funcionalidades

- Login
- Registro de usuários
- Refresh Token
- Roles
- Permissions
- Controle de acesso

## Fluxo

Usuário → Login → JWT → APIs protegidas

## Casos de Uso

- Microsserviços
- APIs corporativas
- Sistemas multiusuário
